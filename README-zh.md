# Aurora Terminal 插件市场

Aurora Terminal 插件的官方索引库。基于 GitHub Actions 和 Pages 构建。

## 如何使用？
插件索引地址为：`https://aurora-terminal.github.io/plugin-registry/registry.json` (请替换为你的实际 GitHub Pages 地址)。

## 开发者相关
请阅读 [开发者指南](./docs/developer-guide-zh.md) 了解如何创建并提交你的插件。详细的 API 信息请参考 [API 参考文档](./docs/api-reference-zh.md)。
