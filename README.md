# Aurora Terminal Plugin Marketplace

Official marketplace for Aurora Terminal plugins. Powered by GitHub Actions & Pages.

## How to use?
The registry index is available at: `https://aurora-terminal.github.io/plugin-registry/registry.json` (Replace with your actual GitHub Pages URL).

## For Developers
Please refer to the [Developer Guide](./docs/developer-guide.md) to learn how to create and submit your plugins. Detailed API information can be found in the [API Reference](./docs/api-reference.md).
